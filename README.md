\# Grove Plumbing — Website Redesign Demo



This is a modern website redesign concept for a local plumbing business.



\## Purpose



To demonstrate how a cleaner design, structured service pages, and a simple booking flow can improve trust, usability, and lead generation.



\## Features



\* Modern, mobile-friendly design

\* Service and service area pages

\* Request service / booking interface (front-end demo)

\* Strong calls to action

\* Local SEO structure



\## Stack



\* Next.js (App Router)

\* TypeScript

\* Tailwind CSS

\* shadcn/ui



\## Notes



This is a front-end demo. Booking and form submissions are not connected to a backend yet.
